# Domani Dashboard Release Mockups

Static dashboard mockups for the Domani release management workflow in the PVS dashboard.

These are planning artifacts only. They are isolated in the Domani landing repo and are not wired into either app.

## Files

- `dashboard-releases-list.png` / `dashboard-releases-list.svg`
- `dashboard-release-detail.png` / `dashboard-release-detail.svg`
- `dashboard-markdown-import.png` / `dashboard-markdown-import.svg`
- `dashboard-conversion-review.png` / `dashboard-conversion-review.svg`
- `generate-dashboard-mockups.js`

## Design Notes

- Matches the current PVS dashboard structure: left sidebar, Domani subnav tabs, rounded white surfaces, purple active states, and dense admin controls.
- Adds a proposed `Releases` tab under `/dashboard/domani`.
- Keeps the dashboard workflow separate from the public release pages.
- Covers four core states: release list, release editor, Markdown import, and conversion review.
